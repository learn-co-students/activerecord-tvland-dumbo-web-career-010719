Mini Arcade!
Created by Joseph Gao
Developed using Ruby and ActiveRecord, along with Sqlite3 database management.

Gems:
gem 'pry'
gem "sinatra-activerecord"
gem "sqlite3"
gem "rake"
gem "rainbow"
gem 'tty-prompt'
gem "tty-font"
gem "table_print"

Setup Instructions:
1. bundle install

To execute:
Use the terminal to type in the following:
1. rake db:migrate
2. rake db:seed
3. ruby run.rb
